# Increment Button

A button with an onClick handler which increments a counter and stores persistent state.

[Demo found here](https://dev.d2sbvvl7jo2app.amplifyapp.com/)

## Instructions

run yarn to build

```console
yarn
```

and

```console
yarn dev
```

to start dev server
